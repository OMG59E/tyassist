# TyAssist

DEngine开发辅助工具

## Usage

### build

```shell
python3 dpexec.py build -c config.yml --log_dir ./logs
```

### compare

```shell
python3 dpexec.py infer -c config.yml --log_dir ./logs
```

### test

```shell
python3 dpexec.py test -c config.yml --log_dir ./logs
```

### demo

```shell
python3 dpexec.py demo -c config.yml --log_dir ./logs
```
